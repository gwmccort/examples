== Useful Docker Commands
[qanda]
Get docker port::
docker port <container>

Get docker-machine ip::
docker-machine ip default

Run container with name::
docker run -d -P --name <name> <image>

Copy file from container::
docker cp -L cotainer:/path/to/file .
