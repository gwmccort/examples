== Example
* type commands
....
> gradle -q --no-daemon --parallel-threads 0 shell
groovy:000> import geb.*
groovy:000> b = new Browser(baseUrl: "http://nile:8008/webquery")
groovy:000> b.go
groovy:000> b.title
b.title
===> PDM WebQuery
....

* load script
....
groovy:000> :l wqTitle.groovy
....

* print page source
....
b.driver.pageSource
....
